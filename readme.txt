Steps to install project:

>copy the php_project_01 in htdocs folder.

>open xampp control panel and start the apache server and mysql database.

>open browser and type localhost.

>type php_project_01.index.php (i.e.,http://localhost/php_project_01/index.php)

>and the project will start.

> and to import sql in phpmadmin

> open phpmyadmin and go to import and select database folder and select file php_project_01.sql
